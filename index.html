Welcome to the iFirmware Page

Overview

What Is iFirmware?

iFirmware is an easy to use application to download the latest iOS firmwares and even older ones. Supports many Apple devices.

How Did This Become?

Well after seeing iH8sn0w's https://github.com/iH8sn0w/sn0wbreeze in action, I had seen that it had a built in downloader for iOS firmwares, so I decided to make one of my own, a standalone iOS firmware downloader called iFirmware.

Compatibility

iFirmware is currently only compatible for 32-Bit arch machines but you could help out with a 64-Bit arch version of iFirmware by re-compiling the source code on your 64-Bit machine and pushing it to the master64 branch.

Please note that when downloading the source code you are downloading the 32-Bit one, hence master being the default, no worries it is the fact that it must be compiled on a 64-Bit machine.
